FAQ

Which AI providers are supported?

The sidebar works with popular providers including OpenAI, Google Gemini, and Claude, so you can choose the models that fit your workflow.

Do I need my own API key?

Yes. You connect your own API keys from the provider you prefer. Keys are stored locally in your browser's extension storage and are only sent to the selected provider.

Is there a free option?

Some providers, like Google Gemini, offer free usage tiers with limits. Check your provider's pricing page for current details.

Where can I report bugs or request features?

Issues and feature requests are tracked on GitHub. Visit the repository's issues page to file a report or share ideas.

API Keys

AI Assistant Sidebar uses your own API keys. One easy way to get started is with a Google Gemini key, which many users can access on a free tier.

Get a free Gemini API key

  • Sign in to your Google account and go to Google AI Studio.
  • Choose to create a new API key in a fresh project.
  • Once the key is generated, copy it and store it somewhere safe. You typically cannot view it again later.

Use your key in the extension

  • Open the extension’s options page from your browser after installation.
  • Select Gemini as your provider and paste your API key into the field provided.
  • Pick the recommended model, save your settings, and you’re done.
